# @githanger/web

GitHanger dashboard app (React + Vite).

## Development

From repo root:

```bash
npm run -w @githanger/web dev
```

Default URL: `http://localhost:5173`.

## Build

```bash
npm run -w @githanger/web build
```

## Notes

- This package is intended for local dashboard development in the monorepo.
- It depends on the local GitHanger API server (`packages/server`).
